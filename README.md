## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Seed

```bash
# seed
$ yarn seed

# seed fresh
$ yarn seed:refresh
```
